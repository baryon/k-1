# 《Audit-Gated Gradient Projection from Information-Curvature Index Subspaces for Continual Learning》阅读理解

## 论文信息

- 原文文件：`./paper/Audit-Gated.pdf`
- 作者：Y. Y. N. Li
- 篇幅：11 页

## 原文主旨

这篇论文把上一篇 $\Gamma^{\mathrm{info}}$ 的层级几何信号，进一步推进到“方向级”的持续学习控制。作者的基本判断是：灾难性遗忘不是因为某一层整体不能动，而是因为新任务梯度碰到了旧任务的重要方向。如果能找到 $\Gamma^{\mathrm{info}}$ 的近零特征方向，也就是“信息曲率接近 0 的方向”，这些方向也许就是相对安全的更新通道。

但作者并没有直接宣称自己恢复了整个参数空间的真实零空间。相反，这篇论文最核心的设计恰恰是“审计”：

- 只在随机抽样的坐标子空间里估计 $\Gamma^{\mathrm{info}}$
- 把同一参数在两个 disjoint batch half 上各估一次
- 比较两次近零子空间是否对齐
- 只有可重复、可复现的参数，才允许进入后续梯度投影

这个机制就是 Criterion A。论文真正想证明的不是“我有精确理论”，而是“我有一个可以在训练前先做可靠性检查的几何代理”。

## 原文内容梳理

### 1. 问题设定：从标量重要性走向方向几何

作者先回顾持续学习的经典方法 EWC。EWC 用 Fisher 对角线给每个参数分配一个重要性系数，本质上是标量保护。论文认为这不够，因为遗忘风险并不只是“哪个参数重要”，更是“参数空间里的哪些方向危险”。

因此，作者希望把前一篇论文里的 $\Gamma^{\mathrm{info}}$ 从“层级风险信号”扩展成“方向级安全通道”。

### 2. 理论接口：把 $\Gamma^{\mathrm{info}}$ 的近零空间看成安全更新方向

在 Euclidean 近似下，$\Gamma^{\mathrm{info}}$ 可以看成熵 Hessian 的归一化形式。若某个方向 $v$ 满足

$$
\Gamma^{\mathrm{info}} v = 0
$$

那么沿这个方向，二阶熵曲率消失。原文的意思是：如果旧任务训练后模型已经接近平稳点，那么这些方向上的更新在二阶上不会显著扰动旧任务的信息结构，所以有希望减少遗忘。

不过作者也反复提醒，这只是“局部、近似、经验成立”的论证，不是全局定理。因此论文没有直接跳到方法，而是先设计审计。

### 3. Criterion A：跨 batch 的近零子空间复现性

Stage 1 的关键是 Algorithm 1。对于每个候选参数：

1. 随机抽样多个坐标子集。
2. 把探测 batch 分成两半。
3. 分别在两半数据上估计 $\Gamma^{\mathrm{info}}$ 的子块。
4. 取其中的 near-null eigenspace。
5. 计算两个 near-null 子空间的主角度对齐程度。

Criterion A 本质上是在问：你看到的“零曲率方向”到底是真结构，还是某一小批数据带来的噪声假象？

论文的门控标准是：

- $A > 0.5$
- 且至少在 $2$ 个独立子集上通过

通过后，该参数才会进入 Stage 2 的投影更新。

### 4. 实验一：GPT-2 上大多数候选参数都能通过审计

实验使用 GPT-2，从 WikiText-2 迁移到 Biomedical Medical QA，关注 h.6-h.11 共 42 个候选参数。结果是：

- 5 个 seed 上，每次都有 $41\text{--}42 / 42$ 个参数通过
- 总通过率 $98.6\%$
- layer norm、attention bias 等参数几乎是满分复现
- 最大的 MLP 权重矩阵通过率稍低，但仍然有 $90\%$

原文要表达的是：$\Gamma^{\mathrm{info}}$ 的近零结构不是偶然噪声，而是在 GPT-2 这些层里广泛存在且相对稳定。

### 5. 实验二：硬投影能减轻遗忘，但强基线仍更强

Stage 2 把新任务梯度投影到审计通过的 near-null 子空间上。主要结果是：

- `free`：遗忘 $+414 \pm 45$
- `gamma along`：遗忘 $+331 \pm 30$
- `gamma random`：遗忘 $+369 \pm 68$
- `EWC`：遗忘 $+217 \pm 33$
- `packnet simplified`：遗忘 $+225 \pm 11$

所以论文得到两个层次的结论：

1. 相比无约束微调，`gamma along` 的确显著降低遗忘，而且新任务困惑度几乎不受影响。
2. 但如果和 EWC、PackNet 这类更强基线比，当前方法还没有赢。

另外一个很重要的对照是 `gamma random`。它和 `gamma along` 在支撑索引和子空间维度上完全匹配，只是方向随机。这样做是为了隔离一个问题：到底是“做了投影”有效，还是“投到了正确的几何方向”才有效？

结果是 `gamma along` 优于 `gamma random`，而 `gamma random` 又优于 `free`，方向上支持作者的几何解释，但 `gamma along` 对 `gamma random` 的差距在 5 个 seed 下还没有达到显著性。

### 6. 实验三：软投影反而不如不投影

论文还试了 soft projection，也就是保留一部分原始梯度。结果 $\alpha = 0.75$ 和 $0.50$ 都比 `free` 更差。作者的解释是：在当前学习率和模型规模下，只保留一部分 null 约束并不足以压住危险方向，真正有效的是硬投影。

### 7. 论文提出了后续路线图

最后一部分作者没有把当前方法包装成终点，而是明确写了三个下一步：

- 用 $\lVert \Gamma^{\mathrm{info}} \rVert$ 先做层选择
- 用 $\lVert \Gamma^{\mathrm{info}} \rVert$ 设计按层学习率
- 用 $\lVert \Gamma^{\mathrm{info}} \rVert$ 连续调节投影强度，而不是现在这种二元 gate

也就是说，这篇论文更像是把“几何信号 + 审计 + 持续学习”三件事先接通。

## 我的理解

我觉得这篇论文最成熟的地方，不是结果绝对值，而是方法学态度。作者清楚知道 $\Gamma^{\mathrm{info}}$ 只是一个近似代理，完全恢复全局零空间在 GPT-2 这种规模下不现实，所以没有直接喊“发现了安全子空间”，而是加上了一个跨 batch 可复现实验，先验证这个局部零空间值不值得信。

这让整篇文章比单纯“用某个张量做投影”要可信得多。Criterion A 的意义在于，它把几何理论变成了一个训练前质量检查工具。换句话说，这篇论文真正的贡献不是某个单独的投影公式，而是“先审计，再使用”的流程。

从效果上看，我的判断比较克制：

- 对比 `free`，结果是正面的，而且有统计显著性。
- 对比 `gamma random`，结果有方向证据，但还不够硬。
- 对比 `EWC/PackNet`，当前版本并不占优。

所以最合理的理解不是“新方法已经超过主流 CL 方法”，而是“作者找到了一个有几何含义、并且可被审计的额外控制维度”。这对研究是有价值的，因为它补的是“方向结构”而不是“再发明一个重要性惩罚项”。

我还会特别强调这篇论文自己承认的限制：它验证的是 index-subspace 的局部近零结构，不是全参数张量的全局零空间证明。因此，阅读时不能把 $A$ 通过理解成“这个参数整体绝对安全”，更准确的说法应该是：在多个抽样坐标子空间里，这个参数出现了可重复的近零方向结构，值得用于近似投影。

如果把它放到整个系列里看，这篇文章承上启下：它继承了上一篇关于 $\Gamma^{\mathrm{info}}$ 的几何解释，又和下一篇 `Neural Null Cones` 一样，都在寻找“零曲率更新通道”。区别是，这篇更偏谨慎的工程代理与审计框架，而不是直接从 Hessian-辛分解给出代数定理。
