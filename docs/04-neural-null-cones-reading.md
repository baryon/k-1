# 《Neural Null Cones》阅读理解

## 论文信息

- 原文文件：`./paper/Neural Null Cones.pdf`
- 作者：Y. Y. N. Li
- 篇幅：14 页

## 原文主旨

这篇论文提出了整个系列里最鲜明的一个几何对象：null cone。作者认为，神经网络损失 Hessian 如果是非定的，那么其中会藏着一类特殊方向：沿这些方向更新，关于正则化 Hessian $H_{\mathrm{reg}}$ 的二阶代价恰好为零。

关键点在于，这些方向不是 Hessian 自身的特征向量，而是

$$
M = H_{\mathrm{reg}}^{-1} J
$$

这个辛算子的实特征向量。只要 $M$ 有实特征对，那么对应特征向量一定满足

$$
e^\top H_{\mathrm{reg}} e = 0
$$

这就是论文的 Spectral Null Cone Theorem。作者把这个定理当成一个新的几何不变量来源：传统 Hessian 谱分析看不到它，因为它根本不是在看 $H$ 的特征结构，而是在看 $H^{-1}J$ 的实谱。

## 原文内容梳理

### 1. 从二维自由能例子出发，展示零曲率方向如何出现

论文先用一个 2 维自由能 Hessian 举例：当 Hessian 进入 Lorentzian/不定区域后，$JG$ 会出现实特征值，于是临界阻尼更新方向满足

$$
ds^\top G\,ds = 0
$$

也就是零二阶代价。作者把它当成高维定理的直观原型：如果二维里已经会出现“光锥式”的零方向，那么神经网络的大维度 Hessian 也可能有类似通道。

### 2. 核心定理：实特征向量必然是 $H_{\mathrm{reg}}$-null

第三部分是论文的数学核心。作者先证明：

- $G^{-1}J$ 对 $G$ 来说是 skew-symmetric
- 若 $M = H_{\mathrm{reg}}^{-1}J$ 有实特征值 $\lambda$ 和实特征向量 $e$
- 则 $e^\top H_{\mathrm{reg}} e = 0$

论文还进一步证明了：

- 不同实特征值对应的实特征向量彼此 $H_{\mathrm{reg}}$-正交
- 它们的线性组合仍然是 $H_{\mathrm{reg}}$-null

原文想强调的是：null cone 不是数值巧合，而是满足前提条件时的代数必然结果。

### 3. 经验发现：神经网络 Hessian 经常真的满足这些前提

作者随后从一个 22 参数 XOR 小网络开始，发现训练过程中：

- Hessian 一直是不定的
- $H_{\mathrm{reg}}^{-1}J$ 一直有实谱
- 每一步都能找到 1 到 6 个 null directions

然后论文把验证扩展到 LeNet-5、GPT-2、ViT-Base、BERT-Base。结果总结得很强：

- 437 个数值实验中，397 个满足定理条件
- GPT-2 的 6/7 层类型有 null cone 结构
- ViT-Base 是 14 个层类型里 13 个成立
- BERT-Base 是 17 个层类型里 15 个成立
- 典型残差能到 $10^{-26}$

这里原文真正想传达的是：null cone 不是 toy phenomenon，而是跨架构、跨模态反复出现的结构。

### 4. 方法一：NCA-SGD，用 null 方向增强优化

论文提出 Null Cone Assisted SGD：

1. 定期算一个局部 $H_{\mathrm{reg}}$
2. 求 $H_{\mathrm{reg}}^{-1}J$ 的实特征方向
3. 过滤掉不满足数值残差要求的方向
4. 把当前梯度分解为 null 部分和剩余部分
5. 只增强 null 部分

在小网络和 ViT-Tiny 上，作者报告：

- ViT-Tiny 上精度比 SGD 高 $+5.7\%$
- 收敛速度快 $6.7\times$
- 如果把同样的增强用到随机方向上，性能反而下降 $-2.8\%$

这个随机方向对照特别关键，因为它说明收益不是来自“多加了一点梯度”，而是来自“加在了零曲率方向上”。

### 5. 方法二：自动搜索 $J$，扩大 null 通道

因为 null direction 来自 $H_{\mathrm{reg}}^{-1}J$，所以 $J$ 的配对方式会影响能找到哪些 null directions。论文进一步做了自动搜索，让 $J$ 去最大化梯度在 null 子空间里的比例。结果 null channel width 从 $18.7\%$ 扩大到 $82.7\%$，提升约 $4.4\times$。

作者想说明的不是“J 可以凭空造出 null directions”，而是：Hessian 本身已经有这些零曲率通道，只是不同配对方式能找到的多少不同。

### 6. 方法三：NCCL，把 null subspace 用到持续学习

后半篇把 null cone 用到 continual learning：

- 在 ViT-Tiny 双头任务里，NCCL 的 forgetting 是 $2.0\%$
- naive fine-tuning 是 $7.9\%$
- EWC 是 $8.8\%$

三任务实验里，作者继续把梯度投影到 $null(H_A) \cap null(H_B)$，发现交集虽然缩小，但没有塌掉，仍能保护旧任务并学习新任务。

在 GPT-2 上，作者又做了 WikiText-2 到 LAMBADA 的迁移。结果是：

- `NCCL_pure` 的遗忘最低
- 但新任务学习容量不足
- 少量混合投影 $\alpha = 0.1$ 能在保护旧任务和学习新任务之间取得折中

这说明 null cone 方法在持续学习里是有效的，但“保护强”和“可塑性强”之间仍然存在 trade-off。

## 我的理解

这篇论文最强的地方，是把“零曲率方向”从一个口号做成了完整闭环：

1. 先给代数定理。
2. 再证明神经网络 Hessian 经验上经常满足定理前提。
3. 再把它做成优化和持续学习算法。
4. 再用随机方向、不同 $J$、多架构实验去说明这不是偶然。

和 `Audit-Gated` 相比，这篇论文更“硬”：它不是从 $\Gamma^{\mathrm{info}}$ 的经验代理出发，而是直接围绕 $H_{\mathrm{reg}}^{-1}J$ 的实谱做文章。因此它的理论味道更强，实验覆盖也更广。

我个人对这篇文章的理解是：

- 它真正的新意不是“又找到一些平坦方向”，而是指出这些方向不是 Hessian 特征向量，而是辛结构诱导出来的实谱方向。
- 这让“零曲率通道”变成一种传统谱分析没有显式看到的几何资源。
- 一旦这个资源存在，就能解释为什么某些更新几乎不付二阶代价，也就自然适合做加速优化或保护旧任务。

不过这篇论文也有几个必须带着看的限制：

1. 大模型实验依赖 50×50 的坐标子 Hessian，不是完整全 Hessian 结论。
2. 很多训练收益还是 preliminary，尤其 ViT-Tiny 训练只用了 2 个 seed。
3. $J$ 的选择既是优势也是不确定性来源。论文说 nullness 是 $H_{\mathrm{reg}}$ 的性质，不是 $J$ 的幻觉；这一点从定理上成立。但在实践中，效果能好到什么程度，确实受 $J$ 搜索策略影响很大。
4. GPT-2 的持续学习实验只有 1 个 seed，结论方向上有意思，但还不够稳。

整体上，我会把这篇论文看成整个系列里最有“发现新结构”意味的一篇。它提出了一个新的观察角度：神经网络的 Hessian 不只是有 sharp/flat、正/负特征值这些老问题，还可能藏着由辛配对揭示出来的零曲率锥结构。只要这个观察在更大规模实验中继续成立，它对优化和持续学习都可能是一个真正新的工具箱。
